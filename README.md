# Project Title

This is a demo Flask app for utilizing the F5 python sdk

## Getting Started

First, clone the repo. Then build and run in a docker container

```
docker build -t unit114-py37 .
docker run -d --rm --name f5demo -p 5000:5000 unit114-py37
```
